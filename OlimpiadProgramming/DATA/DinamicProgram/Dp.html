Динамическое программирование



Динамическое программирование - это метод решения задач путем её разбиения на подзадачи. Например числа Фибоначчи - для вычисление некоторого числа в этой последовательности нужно найти предыдущие.

Задача: Максимальная возрастающая подпоследовательность.

Вам дан массив из n чисел a1...an. Нужно найти числа b1...bk образующие возрастающую подпоследовательность.

Решение: заведем массив dp, в котором все элементы равны 1. Для каждого dpi от 1 до i-1 будем находить максимальную dpj, для которой ai > dpj. Это работает за O(n2).

В динамическом программировании есть алгоритмы оптимизации решения. Например Convex Hull Trick, Meet-in-the-middle.
